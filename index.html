    About

    This is a work-in-progress book on text processing with command line Rust tools.

    Planned list of tools

    • ripgrep
      • ripgrep-all
    • sd
    • frawk
    • hck
      • choose
    • fd
    • rnr
    • huniq
    • bat — for options like -A, -n, etc
    • jql
    • xsv
    • Et cetera

    Rust rewrite of the GNU coreutils won't be covered here. I'm planning to write a separate book for GNU coreutils.
